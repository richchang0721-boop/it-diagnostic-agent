                                                  使用本地模型前請確認
                                                  請選擇你的 Ollama 部署方式，並確認以下項目
                                                  • 已在這台電腦安裝 Ollama（ollama.com）
                                                  • Ollama 服務已啟動（ollama serve）
                                                  • 已下載模型（例如 ollama pull gemma3:12b）
                                                  • 端點設定為 http://localhost:11434（預設值）
                                                  • 內網 Ollama Server 已安裝並啟動
                                                  • 已在設定頁面填入正確的伺服器 IP（例如 http://192.168.1.100:11434）
                                                  • 防火牆已開放 11434 port（或自訂 port）
                                                  • Ollama 已設定允許外部連線（OLLAMA_HOST=0.0.0.0）
                                                  • 已下載所需模型至伺服器
                                                  ⚠️ 注意：若透過 HTTPS 網址（如 GitHub Pages）存取本工具，瀏覽器安全限制可能封鎖 HTTP 本地端點。建議將 index.html 下載後在本機直接開啟，或確保 Ollama Server 有 HTTPS 支援。
                                                  Ollama 不需要 API Key。請確認本機已安裝並啟動 Ollama 服務。
